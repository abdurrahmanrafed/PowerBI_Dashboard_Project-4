# PowerBI_Dashboard_Project-4
A project analyzing 5.3 billion trips across the NSW public transport network (2016–2025)!

Dashboard 1 — Senior Manager View
Designed for strategic & operational decision-making:
· Usage trends across 5 transport modes (Train, Bus, Ferry, Metro, Light Rail)
· Customer satisfaction analysis by mode (2016–2023)
· Card type usage over time (Adult, Concession, Child/Youth, Senior)
· Peak ridership heatmap by month & mode
· Year-on-Year growth tracking

Dashboard 2 — Public View
Designed to be simple, engaging & jargon-free:
· Busiest months to travel
· Fare caps by card type (Daily & Weekly)
· Overall passenger satisfaction gauge (93%!)
· Who is travelling — card type breakdown
· Key facts & positive trends

Tools & Skills used:
• Power BI Desktop
• Power Query (data cleaning & transformation)
• DAX (calculated measures & KPIs)
• Data modeling & relationships
• Dashboard design & UX for two different audiences
